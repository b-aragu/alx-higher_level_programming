My class
________________

- This module contains a class MyList that inherits from list

::
    >>> MyList = __import__('1-my_list').MyList
::

- All elements of the list are assumed to be of type int

::
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
::

- The print_sorted function in MyList returns a sorted list

::
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
::

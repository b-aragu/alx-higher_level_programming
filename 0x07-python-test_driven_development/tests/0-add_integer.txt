# 0-add_integer.py

This file contains a test for a function that add integers

>>> add_integer = __import__('0-add_integer').add_integer

>>> add_integer(1, 2)
3

>>> add_integer(100, -2)
98

you can pass only one arguement , the second one b is already initialized to 98
>>> add_integer(2)
100

>>> add_integer(3)
101

if you pass a float, it is typecasted to an int and then addition is done
>>> add_integer(100.3, -2) 
98
>>> add_integer(20, 1.1)
21

if you pass an arguement that is not an int or float a TypeError is raised
>>> add_integer(4, "school")

	...
TypeError: b must be an integer
>>> add_integer("virgil", 34)
Traceback (most recent call last):
	...
TypeError: a must be an integer
